# Project Overview

This project is a portfolio-grade autonomous UAV navigation system.

The first stage focuses on a 2D reinforcement learning environment where a UAV learns to reach a target while avoiding obstacles.

The later stages will extend the project into:

- dynamic obstacle avoidance
- continuous control
- TD3 reinforcement learning
- Unreal Engine/PX4 simulation
- AirSim-compatible UAV simulation
- robotics-style evaluation metrics

## Main Learning Goals

- Python software engineering
- Reinforcement learning
- UAV path planning
- robotics simulation
- testing and reproducibility
- portfolio-ready documentation
